# Figure Audit

Read-only audit of every `\begin{figure}...\end{figure}` float in the book.
Generated by `scripts/audit_figures.sh` plus manual caption/style review.

**Status: Phase 2, Phase 3, and Phase 4 executed.** All raster conversions,
wrapper additions, font bumps, caption rewrites, and palette migrations have
been completed. This document was the input to the migration and now also
records the final post-migration state at the top of each section.

Companion files:

- `FIGURE_STYLE_GUIDE.md` - the style contract every figure must satisfy.
- `FIGURE_AUDIT_RAW.txt` - script output: one pipe-delimited row per figure.
- `FIGURE_REFS.txt` - script output: one row per label with `\ref`/`\Cref` count.

### What changed

- **Phase 2 (raster -> TikZ)**: 5 rasters (`ch01_cost_latency_throughput`,
  `ch01_rag_drift_control`, `ch01_ishtar_arch_main`,
  `ch03_cloud_native_deployment`, `ch03_ishtar_infra`) were hand-ported to
  TikZ using the `llm/*` style macros. Legacy PNGs were moved to
  `images/_archive/` so they remain recoverable until print review.
- **Phase 3a (wrappers)**: `fig:ch01_llmops_legend` and
  `fig:ch01_llmops_quick_checklist` are now wrapped in `llmfigbox`. The
  checklist's `tcolorbox` frame was recolored to the `llmblue` palette.
- **Phase 3b (typography)**: `\tiny` and `\scriptsize` inside
  `fig:ch03_deploy_patterns_balanced`, `fig:ch10_metrics_dashboard`,
  `fig:ch11_privacy_lifecycle`, and `fig:ch11_escalation_tree` were bumped
  to `\footnotesize` (42 lines total). The five new TikZ ports still use
  `\scriptsize` inside dense node callouts - this is permitted by section 7
  of `FIGURE_STYLE_GUIDE.md` ("legends or axis ticks when space is tight").
- **Phase 3c (captions)**: 8 flagged captions were rewritten so the first
  sentence is an explicit reader takeaway. Affected labels:
  `fig:ch02_llm_scale`, `fig:ch06_ishtar_scaling_timeline`,
  `fig:ch07_quantization_throughput`, `fig:ch10_cicd_decision_flow`,
  `fig:ch11_bias_audit_cycle`, `fig:ch11_privacy_lifecycle`,
  `fig:ch11_escalation_tree`, `fig:ch12_ishtar_metrics_evolution`.
- **Phase 3d (palette)**: `scripts/migrate_figure_palette.py` rewrote 25
  figures across 9 chapters, mapping each local `\definecolor{...}{RGB}{...}`
  to the nearest book palette slot (`llmblue`, `llmorange`, `llmgreen`,
  `llmpurple`, `llmgray`) and then replacing every occurrence of the local
  name inside the figure block. 136 `\definecolor` lines were deleted; the
  one remaining local redefinition (in `ch05-observability.tex`) was deleted
  by hand because it collided with the global palette name.
- **Phase 4 (build)**: a full `latexmk -pdf book.tex` run reproduced the
  exact same warning counts as the Phase 0 baseline; no new warnings were
  introduced. The resulting `book.pdf` is ~500 KB smaller because the five
  rasters are no longer embedded.

### Residual items (known, out of scope for this pass)

- Some of the 55 pre-existing TikZ/PGFPlots figures still do not use the
  `llm/box`/`llm/arrow`/`llm/chart`/`llm/cycle` *structural* styles
  (they only now share the palette). Converting their node and arrow
  specifications to `llm/*` styles is a lighter-touch refactor for a future
  pass; the style contract is already enforceable for new figures.
