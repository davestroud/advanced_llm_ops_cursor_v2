# Complete File List for Overleaf Upload

## Essential Files for Compilation

This document lists all files required to compile the LaTeX monograph "Advanced Large Language Model Operations" in Overleaf.

---

## Root Directory Files (5 files)

These files must be uploaded directly to the project root:

1. **`book.tex`** - Main document file (entry point for compilation)
2. **`SNmono.cls`** - Springer Nature monograph document class (required)
3. **`svind.ist`** - Index style file for `makeindex` (required for index generation)
4. **`macros.tex`** - Custom LaTeX macros and commands (required)
5. **`references.bib`** - Bibliography database (required for citations)

---

## Images Directory

If you have image files (`.png`, `.jpg`, `.pdf`), create an `images/` folder in Overleaf and upload them there. The `book.tex` file is configured to look for images in `./images/` via `\graphicspath{{./}{images/}}`.

**Note:** Currently, all `\includegraphics` commands in the source files are commented out, so no images are required for compilation.
